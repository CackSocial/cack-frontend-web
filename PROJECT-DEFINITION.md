# Project Definition: SocialConnect

## Project Overview

**SocialConnect** is a lightweight social networking platform designed to facilitate real-time content sharing and interpersonal connections. The platform focuses on visual and textual expression, allowing users to curate their feeds through follows and stay updated on global conversations via trending topics.

Users have unique usernames and they will login using username and password, no need for email.

---

## Core Features

## User Connections

- **Following System:** Users can follow other profiles to subscribe to their content updates.
- **Timeline:** A centralized, chronological feed that aggregates posts exclusively from the accounts a user follows.

##  Content Creation

- **Versatile Posting:** Users can share updates consisting of:
	- Plain text.
	- images.
	- Hybrid posts (Text + Image).
- **Categorization:** Support for **Tags** (hashtags) within posts to organize content and improve searchability.

## Discovery

- **Trending Tags Section:** A dynamic sidebar or dashboard that highlights the most frequently used tags across the platform in real-time, allowing users to jump into viral conversations.

## Private Communication

- **Direct Messaging (DM):** A private one-on-one communication channel.
- **Rich Media Support:** DMs support text, images, or a combination of both, ensuring the private experience is as robust as public posting.
